## 二、模块详细说明

### 1. Backbone: ResNet-50

```
┌──────────────────────────────────────────────────────────────┐
│                    ResNet-50 架构                            │
│                                                              │
│  Stage     Output Size    Block Type    Channels             │
│  ─────────────────────────────────────────────────           │
│  conv1     112×112        7×7 conv      64                   │
│  maxpool   56×56          3×3 maxpool   64                   │
│  layer1    56×56          Bottleneck×3  256                   │
│  layer2    28×28          Bottleneck×4  512   ← C3 for FPN   │
│  layer3    14×14          Bottleneck×6  1024  ← C4 for FPN   │
│  layer4    7×7            Bottleneck×3  2048  ← C5 for FPN   │
│                                                              │
│  特点:                                                      │
│  - 预训练权重 (ImageNet)                                    │
│  - 提取多尺度特征                                           │
│  - FPN使用 C3, C4, C5 三层特征                              │
└──────────────────────────────────────────────────────────────┘
```

### 2. Feature Pyramid Network (FPN)

```
┌──────────────────────────────────────────────────────────────┐
│                    FPN 结构                                  │
│                                                              │
│     C5 (2048ch) ──────→ Lateral Conv ──→ P5 (256ch)         │
│                          │                ↑                  │
│                          │                │ Upsample 2×      │
│     C4 (1024ch) ──────→ Lateral Conv ──→┼→ P4 (256ch)       │
│                          │                ↑                  │
│                          │                │ Upsample 2×      │
│     C3 (512ch)  ──────→ Lateral Conv ──→┼→ P3 (256ch)       │
│                                           │                  │
│                                           └─→ Output Conv    │
│                                                              │
│  作用:                                                      │
│  - 融合多尺度特征                                            │
│  - 增强小目标检测能力                                        │
│  - 统一输出通道数为256                                       │
└──────────────────────────────────────────────────────────────┘
```

### 3. Region Proposal Network (RPN)

```
┌──────────────────────────────────────────────────────────────┐
│                    RPN 结构                                  │
│                                                              │
│  输入: P3 [B, 256, H/8, W/8]                                │
│                                                              │
│  ┌──────────────────────────────────────────────────────┐    │
│  │  Anchor Generation (每位置生成9个锚框)               │    │
│  │                                                      │    │
│  │  Scales: [32, 64, 128]                              │    │
│  │  Ratios: [0.5, 1.0, 2.0]                            │    │
│  │                                                      │    │
│  │  每个位置: 3 scales × 3 ratios = 9 anchors           │    │
│  └──────────────────────────────────────────────────────┘    │
│                                                              │
│  ┌──────────────────────────────────────────────────────┐    │
│  │  预测头                                              │    │
│  │                                                      │    │
│  │  P3 ─→ Conv 3×3, 256ch ─┬─→ Conv 1×1 → cls [9]    │    │
│  │                          │                           │    │
│  │                          └─→ Conv 1×1 → bbox [36]   │    │
│  │                                                      │    │
│  │  cls: 前景/背景分类 (每anchor一个得分)               │    │
│  │  bbox: 边界框偏移 (每anchor 4个偏移值)              │    │
│  └──────────────────────────────────────────────────────┘    │
│                                                              │
│  输出: Proposals (经过NMS筛选的候选区域)                     │
└──────────────────────────────────────────────────────────────┘
```

### 4. BBox Head (检测头)

```
┌──────────────────────────────────────────────────────────────┐
│                    BBox Head 结构                            │
│                                                              │
│  输入: ROI Features [N, 256, 7, 7]                          │
│                                                              │
│  ┌──────────────────────────────────────────────────────┐    │
│  │  Shared FC Layers                                    │    │
│  │                                                      │    │
│  │  Flatten: 256×7×7 = 12544                           │    │
│  │  FC: 12544 → 1024                                    │    │
│  │  ReLU                                                │    │
│  │  FC: 1024 → 1024                                     │    │
│  │  ReLU                                                │    │
│  └──────────────────────────────────────────────────────┘    │
│                          │                                   │
│              ┌───────────┴───────────┐                       │
│              │                       │                       │
│              ▼                       ▼                       │
│  ┌─────────────────────┐  ┌─────────────────────┐           │
│  │ Classification      │  │ BBox Regression     │           │
│  │                     │  │                     │           │
│  │ FC: 1024 → 3        │  │ FC: 1024 → 12       │           │
│  │ (bg, subject, obj)  │  │ (3 classes × 4)     │           │
│  └─────────────────────┘  └─────────────────────┘           │
│                                                              │
│  输出:                                                       │
│    - cls_scores: [N, 3] 类别得分                            │
│    - bbox_preds: [N, 12] 边界框偏移                         │
└──────────────────────────────────────────────────────────────┘
```

### 5. Interaction Classifier (交互分类器)

```
┌──────────────────────────────────────────────────────────────┐
│              Interaction Classifier 结构                     │
│                                                              │
│  输入:                                                       │
│    - Subject Box: 主体边界框                                │
│    - Object Box: 客体边界框                                 │
│    - P3 Features: FPN特征图                                 │
│                                                              │
│  ┌──────────────────────────────────────────────────────┐    │
│  │  Subject Feature Extraction                          │    │
│  │                                                      │    │
│  │  Subject Box ─→ ROI Align ─→ [1, 256, 7, 7]        │    │
│  │                    │                                 │    │
│  │                    ▼                                 │    │
│  │              Flatten                                 │    │
│  │              FC: 12544 → 512                         │    │
│  │              ReLU + Dropout                          │    │
│  │                    │                                 │    │
│  │                    ▼                                 │    │
│  │              s_feat [512]                            │    │
│  └──────────────────────────────────────────────────────┘    │
│                                                              │
│  ┌──────────────────────────────────────────────────────┐    │
│  │  Object Feature Extraction (同上)                    │    │
│  │                    │                                 │    │
│  │                    ▼                                 │    │
│  │              o_feat [512]                            │    │
│  └──────────────────────────────────────────────────────┘    │
│                                                              │
│  ┌──────────────────────────────────────────────────────┐    │
│  │  Spatial Relation Encoding                           │    │
│  │                                                      │    │
│  │  相对位置计算:                                       │    │
│  │    dx = (obj_cx - sub_cx) / sub_w                   │    │
│  │    dy = (obj_cy - sub_cy) / sub_h                   │    │
│  │    dw = log(obj_w / sub_w)                          │    │
│  │    dh = log(obj_h / sub_h)                          │    │
│  │                                                      │    │
│  │  [dx, dy, dw, dh] ─→ FC: 4 → 64 → 128              │    │
│  │                          │                           │    │
│  │                          ▼                           │    │
│  │                    spatial [128]                     │    │
│  └──────────────────────────────────────────────────────┘    │
│                                                              │
│  ┌──────────────────────────────────────────────────────┐    │
│  │  Fusion & Classification                            │    │
│  │                                                      │    │
│  │  Concat: [s_feat, o_feat, spatial] = [1152]        │    │
│  │                                                      │    │
│  │  FC: 1152 → 512                                      │    │
│  │  ReLU + Dropout                                      │    │
│  │  FC: 512 → 256                                       │    │
│  │  ReLU                                                │    │
│  │  FC: 256 → 17 (interaction classes)                  │    │
│  │                                                      │    │
│  │  输出: interaction_logits [17]                       │    │
│  └──────────────────────────────────────────────────────┘    │
│                                                              │
│  17种交互类型:                                               │
│  grasp, contain, lift, open, lay, sit, support,             │
│  wrapgrasp, pour, move, display, push, listen,              │
│  wear, press, cut, stab                                     │
└──────────────────────────────────────────────────────────────┘
```

## 三、训练策略

### 损失函数

```
Total Loss = RPN Loss + Detection Loss + Interaction Loss

┌──────────────────────────────────────────────────────────────┐
│  RPN Loss                                                   │
│                                                             │
│  L_rpn = BCE(cls_logits, rpn_targets)  # 前景/背景分类     │
│        + SmoothL1(bbox_deltas, gt_deltas)  # 边界框回归    │
└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│  Detection Loss                                             │
│                                                             │
│  L_det = CrossEntropy(cls_scores, gt_labels)  # 3类分类   │
│        + SmoothL1(bbox_preds, gt_boxes)  # 边界框精细回归  │
└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│  Interaction Loss                                           │
│                                                             │
│  L_int = CrossEntropy(interaction_logits, gt_interaction)  │
└──────────────────────────────────────────────────────────────┘
```

### 优化器

```
- Optimizer: Adam
- Learning Rate: 
    - Backbone: 1e-5 (0.1x)
    - New Layers: 1e-4
- Weight Decay: 1e-4
- Scheduler: CosineAnnealingLR (T_max=100, eta_min=1e-6)
```

## 五、使用方法

### 训练

```bash
cd iagnet_app/annotation
python annotation/train_annotation.py --config annotation/config_annotation.yaml
```

### 推理

```python
from annotation_model import build_annotation_model

model = build_annotation_model(num_interactions=17)
checkpoint = torch.load('checkpoints/annotation/best.pt')
model.load_state_dict(checkpoint['model_state_dict'])
model.eval()

# 推理
with torch.no_grad():
    results = model(image_tensor)
    
subject_boxes = results['subject_boxes']
object_boxes = results['object_boxes']
interaction_logits = results['interaction_logits']
```

### 使用标注工具

```bash
cd iagnet_app/annotation
streamlit run annotation_tool.py
```

### 查看日志
```bash
tensorboard --logdir ./checkpoints/annotation/logs
```
